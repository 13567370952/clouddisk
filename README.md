#快速向导
<hr>
**(1)如何得到 `CloudDisk`**
  * 使用`maven`下载
```maven 
<dependency>
	<groupId>com.dounine</groupId>
	<artifactId>clouddisk</artifactId>
	<version>x.x.x</version>
</dependency>
 ```
  * 从`github`中下载并编译
```
git clone https://github.com/dounine/clouddisk.git
mvn clean package
```
